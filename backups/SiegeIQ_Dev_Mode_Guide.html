SiegeIQ — Developer Mode

Test the Free, Pro & Squad experience without paying

What it is

Developer Mode is a hidden, browser-only panel that lets you preview the site exactly as a Free, Pro, or Squad member would see it — instantly, with no payment and no test accounts. It changes only what your browser shows. It never touches real billing, your Stripe account, or other visitors.

Opening Developer Mode

  • Keyboard: press Ctrl + Shift + D (Windows) or Cmd + Shift + D (Mac). Press again to close.
  • No keyboard (phone/tablet): add ?dev=1 to the address, e.g. https://siegeiq.gg/?dev=1
  • The panel appears in the bottom-left corner of the screen.

The panel

  • View site as: Free · Pro · Squad
  • Free reviews used: 0 · 1 · 2 · 3
  • Exit dev mode: returns you to your real account
  • A status line shows what you are currently simulating.

Each choice reloads the page so every part of the site updates correctly.

Test the FREE plan (3 reviews / month)

  1. Open Dev Mode and click Free.
  2. Click 0 under “Free reviews used” — you now have all 3 reviews available.
  3. Go to the Upload page. The Analyze button works normally.
  4. Reopen Dev Mode and click 3. The site now behaves as if all 3 free reviews are used: the Analyze button becomes “UPGRADE TO ANALYZE” and the upgrade wall appears.
  5. Try 1 and 2 to see the in-between states.

Test the PRO plan

  1. Open Dev Mode and click Pro. The page reloads.
  2. The badge at the top-right shows PRO · DEV.
  3. Reviews are unlimited — no upgrade wall.
  4. Pro-only interface unlocks — including the “Ask SiegeIQ about this clip” follow-up chatbot on the dashboard and premium AI voice coaching.
  5. The Pricing page reflects a Pro subscriber.

Test the SQUAD plan

  1. Open Dev Mode and click Squad. The page reloads.
  2. The badge shows SQUAD · DEV.
  3. Everything in Pro, plus the team / Squad features.

Returning to normal

Open Dev Mode and click “Exit dev mode (use real account)”. The override clears and the site behaves according to your real signed-in account.

What Dev Mode does & does NOT do

Dev Mode DOES

Simulate the front-end experience for each tier — what each plan sees: upgrade walls, plan badge, unlocked toggles, the clip chatbot, and the 3-review limit.

Dev Mode does NOT

Grant real backend entitlements. Server-gated features — actual ElevenLabs AI voice audio, real clip-processing limits, and true subscription status — still follow your real login.

So “Pro” in Dev Mode shows the Pro interface, but a server-enforced Pro feature only fully works if your real account is Pro (or a demo account). For true end-to-end paid testing — real checkout plus backend unlock — use Stripe TEST mode with test cards and a backend test/demo account.

Resetting the free-review counter

Inside the panel, click 0 under “Free reviews used” to instantly restore all 3 free reviews. (Once dev mode is active you can also press Shift + D anywhere as a shortcut — this is now disabled for normal visitors so the free limit can’t be abused.)

Good to know

  • Dev Mode is invisible to normal visitors — there is no on-screen button; it opens only with the shortcut or the ?dev=1 URL.
  • It is stored only in your browser (localStorage key siegeiq_dev_plan).
